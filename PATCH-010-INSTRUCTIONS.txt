What this patch changes:
- Coach ID is entered before the six-digit Security PIN.
- Invalid credentials use one generic error and do not reveal which field was wrong.
- PBKDF2 + Salt verification remains in place; no plain-text PIN is added.
- Five failed attempts still lock Coach access for fifteen minutes.
- Local and Firebase lock states use the stricter active state.
- Coach login persists across refresh and reopening the PWA.
- There is no automatic session timeout.
- Manual Logout clears the persistent Coach session.
- Member Login and Member PIN are unchanged.
