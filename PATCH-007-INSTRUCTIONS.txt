3. Commit และ Deploy ขึ้น Vercel
4. ปิดและเปิด CLOB ใหม่ หากยังเห็นเวอร์ชันเก่าให้รีเฟรชหนึ่งครั้ง

สิ่งที่เพิ่ม:

- PIN ตรวจสอบด้วย PBKDF2-SHA256 + Salt โดยไม่เก็บ PIN ตัวเลขตรง ๆ ใน JavaScript
- กรอกผิด 5 ครั้ง ล็อก 15 นาที พร้อมใช้ Firebase state และ local fallback
- Coach อัปโหลด ครอป และเปลี่ยนรูปโปรไฟล์ได้
- รูปเก็บใน Firebase Storage: coaches/0409/profile/
- ข้อมูล Coach เก็บใน Realtime Database: clob/coaches/0409
- Dashboard โหลดรูปและข้อมูล Coach จาก Firebase พร้อม local cache
- ไม่มีเมนูเปลี่ยน Coach PIN

ส่วนที่ไม่เปลี่ยน:

หมายเหตุด้านความปลอดภัย:
