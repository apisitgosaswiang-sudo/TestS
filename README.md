# Workout Tracker V2.1

เพิ่มในเวอร์ชันนี้:
- Trainer Dashboard
- Summary Cards
- Quick Actions
- Customer Dashboard Header
- Daily Check-in พร้อม Emoji Mood
- Dark Fitness UI แบบ Active / Professional

วิธี Deploy:
1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub root
3. Commit เข้า branch main
4. Vercel Deploy อัตโนมัติ

ไม่ต้องตั้งค่า Firebase เพิ่ม หาก Anonymous Authentication และ Realtime Database Rules เปิดไว้แล้ว
